## Features

- Create, read, update, and delete tasks with fields such as title, description, status, due date, and reminder
- Group tasks into projects and assign multiple tags for flexible categorization
- Invite teammates to a project with role‑based permissions (owner, editor, viewer)

## Quick Start

### Prerequisites

- Python 3.11+
- Node.js 18+ (for frontend)

### Backend

```bash
cd app
pip install -r requirements.txt
uvicorn main:app --reload
```

API available at: `http://localhost:8000`
Interactive docs: `http://localhost:8000/docs`

### Frontend

```bash
npm install
npm run dev
```

Frontend at: `http://localhost:5173`

### Docker

```bash
docker build -t task_tracker .
docker run -p 8000:8000 task_tracker
```
